# 🍎 Apple Supply Chain & Logistics Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze Apple’s global supply chain and logistics operations. It provides insights into supplier performance, inventory management, shipment tracking, and customer revenue trends.

## 🎯 Objectives
- Monitor supply chain KPIs
- Analyze supplier lead time and unit cost
- Track inventory and defective units
- Monitor shipment delays and carrier performance
- Analyze customer profitability and revenue channels
